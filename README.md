# omeraydemir.github.io

Ömer Aydemir’in kişisel portföy sitesi. GitHub Pages üzerinden repository kökünden yayınlanır.

## Amaç ve teknoloji

Bağımlılıksız statik bir site: semantic HTML5, modern CSS ve gerektiği kadar vanilla JavaScript. Framework veya build adımı yoktur; renk ve tipografi kararları `styles.css` içindeki CSS custom property’lerde tutulur (bkz. `content/design-spec.md`).

## Yerel çalıştırma

```bash
python3 -m http.server 8000
```

Ardından tarayıcıda `http://localhost:8000` adresini açın.
