Featured work

Open-source Raycast extension

Folder Scope

I built a Raycast extension for searching text inside the folder open in Finder without switching to a separate search application. The search flow is designed to be fast, cancellable and local-first.

  • Detects the active Finder folder or selection.
  • Streams cancellable search results as you type.
  • Supports plain-text and regular-expression searches.

Searches run on the device; the network is used only to download bundled ripgrep with SHA-256 verification when needed.

Development approach I used AI tools during ideation and implementation while retaining responsibility for reviewing and validating architecture, security, behavior, testing and quality decisions.
